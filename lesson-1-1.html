🎯 Learning Objective

I can solve one-step equations using inverse operations and check whether my solution makes the equation true.

📚 Learn the Strategy

Big idea: To solve an equation, isolate the variable and keep both sides balanced.

x + 5 = 12
1. Since 5 is added to x, use the inverse operation: subtract 5.
2. Do the same operation to both sides.
3. x = 7 because 7 + 5 = 12.
